### 🔒 [Data Anonymization Pipeline](https://github.com/saintparish4/data-anon-pipeline) — Privacy-First ML Infrastructure
**Python • spaCy • Pandas**

Privacy-preserving data pipeline implementing multi-strategy PII detection (regex + NER) and 4 anonymization techniques. Achieved k-anonymity of 10+ while preserving 92% data utility for ML training.

**Impact:**
- ✅ K-anonymity of 10+ achieved
- ✅ 92% data utility preserved for ML training
- ✅ <5% re-identification risk across 6,500+ test records
- ✅ GDPR/CCPA compliance validation with automated attack simulation

```
🔐 Privacy Techniques
├─ Hashing (irreversible anonymization)
├─ Redaction (sensitive field removal)
├─ Generalization (data aggregation)
└─ Pseudonymization (reversible tokenization)
```
